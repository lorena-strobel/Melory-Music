# 🎸 Melory Music - Sistema de Gestão de Inventário

O **Melory Music** é uma aplicação Ruby on Rails desenvolvida para gerenciar o estoque de instrumentos musicais. O projeto foca em boas práticas de desenvolvimento, incluindo validações de regras de negócio, tratamento de exceções e internacionalização (I18n).

---

##  Tecnologias utilizadas

* **Linguagem:** Ruby 3.2.2
* **Framework:** Ruby on Rails 7.2.3
* **Banco de Dados:** PostgreSQL
* **Sistema Operacional:** Linux Ubuntu
* **Editor:** VS Code
